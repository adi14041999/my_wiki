# Matplotlib

[Matplotlib](https://matplotlib.org/) is a comprehensive library for creating static, animated, and interactive visualizations in Python.

Machine learning is driven by quantitative data. As ML Engineers we need to tailor our choice of algorithms, models and evaluation techniques to the available data.

Understanding the features of a given dataset, along with the ranges of values and shapes of their distributions is best done with visualizations.

Matplotlib is the most commonly used Python library for creating such visualizations. These visualizations can be embedded into the output of Python Notebook cells, making it particularly useful while running experiments. 

## Types of plots

Refer to the [relevant section of the Matplotlib documentation](https://matplotlib.org/stable/plot_types/index.html#plot-types) for the types of plots that can be rendered.

Each example in the gallery links to a simple code sample demonstrating how to render the visualization.
